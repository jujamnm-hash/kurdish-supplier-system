## 🔧 ڕێگای بەکارهێنان

### ١. دامەزراندن
```bash
git clone https://github.com/jujamnm-hash/kurdish-supplier-system.git
cd kurdish-supplier-system
```

### ٢. کرانەوە
فایلی `index.html` بکرەرەوە لە براوزەر، یان:
```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .
```

### ٣. بەکارهێنان
1. **تۆمارکردن** — بڕۆ بۆ `register.html` و ئەکاونتی دابینکەر دروست بکە
2. **کالا زیاد بکە** — داشبۆردەوە، کالاکانت زیاد بکە
3. **پۆلبندی خۆکار** — سیستەمەکە خۆی پۆلبندی دەکات
4. **ریکلام** — کالاکانت لە پەیجی سەرەکی دەردەکەوێت
